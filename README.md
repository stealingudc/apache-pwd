# apache-pwd
Starts an Apache2 server in $(pwd). GNU/Linux & OSX only.
## Installation
Global:
```
$ git clone https://github.com/stealingudc/apache-pwd.git .
$ sudo mv ./apache-pwd.sh /usr/local/bin/apache-pwd
```
Local:
```
$ git clone https://github.com/stealingudc/apache-pwd.git .
```

## Usage
```
$ apache-pwd [port]
```
Locally, use `./apache-pwd [port]` instead.
